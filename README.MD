# Presidential Heights Analysis

### Executive Summary

This project is a beginner's project that explores the presidential heights of United States Presidents


### Key Insights:

* **Finding 1**  [Most president have an average height of around 180 centimeters or six feet]
* **Finding 2** [The shortest president recorded in history is James Madison who has a height of 163 centimeters]
* **Finding 3** [The tallest president recorded in history is Abe Lincoln and Lyndon B. Johnson with a height of 193 centimeters]

### Tech Stack

* **Language**: Python
* **Libraries**: Pandas (Data Manipulation), Matplotlib/Seaborn (Visualization), Numpy (Numerical Analysis)

### How to Run

1. Clone This Repo
2. Install Dependencies
3. Open 'notebooks/Presidential_Height.ipynb' to view the analysis.



You can find the source data located within the repo it is called presidential_height.csv
